Base project I use to start my app engine projects.

Usage
=====
Getting started:

    npm install

Make sure goapp is in your paths.

Run dev version:

    npm run serve

Run prod version:

    npm run serve-prod

Deploy to app engine

    npm run deploy

Other commands, see package.json. Includes: doc, rollback, fmt, and test
